## Contributing to My Sweet Home

We love your input! We want to make contributing to this project as easy and transparent as possible, whether it's:

- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features

### Pull Request Process
1. Fork the repo and create your branch from main.
2. Ensure your code adheres to C++98 standards and follows the existing Design Patterns.
3. If you've added code that should be tested, add tests.
4. Ensure the test suite passes (ctest).
5. Issue that pull request!

### Any questions?
Feel free to contact the maintainers or open a discussion.
